# u27-check Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first open-source Unit27 CLI package, `u27-check`, for deterministic local launch pre-flight checks.

**Architecture:** A Node CLI orchestrates focused modules for package-manager detection, process execution, runtime probing, HTML/CTA parsing, observations, and report output. The first release is Node/web-first and writes both terminal output and the two required files under `u27/`.

**Tech Stack:** Node.js ESM, Node test runner, Cheerio, picocolors, MIT license.
